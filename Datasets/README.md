# Baixe e extraia o LoDoPaB-CT Dataset

**https://zenodo.org/records/3384092**

**Salve os arquivos ZIP nos diretóriose e siga as instruções descritas dentro de cada pasta.**

Acesse as pastas e siga as instruções de cada uma:

Teste: ```.../Datasets/test```

Treino: ```.../Datasets/train```

validação: ```.../Datasets/validation```


## A partir daqui o código irá referenciar e criar as pastas automaticamente

**Quando iniciar a execução, o código irá converter as imagens em .bmp**

Pastas dos imagens convertidas:

**Conjunto de testes**

```.../Datasets/test/ground_truth_test/GT_test_images_bmp```

```.../Datasets/test/observation_test/OBS_test_images_bmp```

**Conjunto de treino**

```.../Datasets/train/ground_truth_train/GT_train_images_bmp```

```.../Datasets/train/observation_train/OBS_train_images_bmp```

**Conjunto de validação**

```.../Datasets/validation/ground_truth_validation/GT_validation_images_bmp```

```.../Datasets/validation/observation_validation/OBS_validation_images_bmp```
